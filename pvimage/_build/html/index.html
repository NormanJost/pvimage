a_BoW.BoW(filepath, N=64, csv_path='./', clusters=4, dist=None, pca=False, global_vec_len=125, haralick=False, custom=True)¶

This function constructs a bag of visual words model using local features such as KAZE, or FAST.

Authors: Ben (bgp12) Args:

filepath:str, a path to the top level directory for the images,

which can then be contained in subdirectories

N:int, the number of features the count in the histogram csv_path:str, where to save the output .csv file clusters:int, the number of clusters to output, e.g. 2 (good, bad) dist:float, optional, use distance rather then cluster number for HC. pca:bool, use PCA on the bag or words or not global_vec_len:int, length to make custom descriptor haralick:bool, use Haralick features or not. custom:bool, use custom descriptor or not.

a_BoW.global_feat(img, length_mult=2)¶

This function returns a feature that represents the row-wise sum of intensities of an image.

Authors:

Ben (bgp12)

Args:
img

A NumPy greyscale (0-255) image array

length_mult

The number of rows to be combined; if the image is 50x50, then a factor of 2 will cause the function to return a 25x1 feature vector.

Returns:

NumPy array: a rows//length_multx1 feature vector

a_BoW.haralick_feature_extraction(images, image_array=True)¶

This function finds the Haralick features from the image(s) for ML classification Notes:

Authors, Ahmad

Args:
images (str): directory where the images are located

or an image numpy array

image_array (bool): True(default), when the numpy object of image is passed

false, when the path of the image(s) is passed

Returns:
Four np arrays, feature vectors for all images in direction:

  1. 0 degrees

  2. 45 degrees

  3. 90 degrees

  4. 135 degrees

  5. list of filenames and class
